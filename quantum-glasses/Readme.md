# Quantum Glasses 👓🔮

A visualization tool for single-qubit rotations on the Bloch sphere using **Qiskit** and **Tkinter**.

## Features
- Visualize X, Y, Z, H, S, T, and rotation gates
- Easy GUI built using Tkinter
- Rotation angles with buttons (π/4, π/2, π, etc.)
- Bloch sphere visualization using `visualize_transition`

git clone https://github.com/sauryaa/quantum-glasses
cd quantum-glasses
